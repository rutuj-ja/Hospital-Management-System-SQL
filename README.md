# 🏥 Hospital Management System (SQL Server)

## 📌 Overview

The Hospital Management System is a SQL Server database project designed to manage hospital operations such as departments, doctors, patients, appointments, treatments, and billing.

This project demonstrates practical implementation of SQL concepts including database design, joins, stored procedures, functions, triggers, views, reports, CTEs, subqueries, and indexing.
